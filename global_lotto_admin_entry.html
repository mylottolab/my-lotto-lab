🌍 해외복권 관리자 수동입력

자동수집(NY Open Data / 스크래핑) 실패 시, 또는 유로밀리언스처럼 아직 자동화가 안 된 게임의 비상 입력용 화면입니다.

관리자 키를 입력하면 현재 표시 중인 데이터를 여기서 바로 확인할 수 있어요.

📌 공식 사이트 참고창 열기 (화면 오른쪽에 자동 배치):

① 확정 추첨결과 입력

미국 복권(파워볼·메가밀리언스) 전용 - 일시불 수령 시 금액입니다. 유로밀리언스는 이 개념이 없어 자동으로 비워집니다.

1등(잭팟)은 위 "1등 당첨금" 칸에 이미 입력하셨으니 여기선 2등부터만 입력하시면 됩니다. 해당 없는 등수는 비워두세요.

등수 맞춘 조합 1인당 당첨금 당첨자 수

② 실시간 잭팟(예상액) 입력

미국 복권 전용. 유로밀리언스는 자동으로 비워집니다.
